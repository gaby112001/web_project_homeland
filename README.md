# Homeland - Proyecto Sprint 5

## Descripción

Este proyecto consiste en el desarrollo de una página web responsiva basada en un diseño proporcionado en Figma.

La página presenta una galería de ciudades natales de empleados, enfocándose en una correcta adaptación a diferentes tamaños de pantalla (desktop, tablet y mobile).

---

## Tecnologías utilizadas

- HTML5
- CSS3
- Metodología BEM
- Flexbox
- CSS Grid
- Media Queries (Responsive Design)

---

## Responsive Design

El sitio fue desarrollado siguiendo un enfoque **desktop-first**, adaptándose a los siguientes rangos:

- Desktop: 1280px en adelante
- Tablet: hasta 1024px
- Mobile: hasta 544px

Se verificó el comportamiento en múltiples resoluciones intermedias para evitar errores de layout.

---

## Características

- Diseño fiel al prototipo de Figma
- Tipografía Inter cargada localmente
- Uso de Grid y Flexbox para layout
- Imágenes optimizadas
- Estructura semántica en HTML
- Organización de estilos con BEM
